# Installation

```bash
$ touch .env(.dev) (Fill with Example)
$ npm install
```

## Example

```

USER = user
DATABASE = database
PASSWORD = password
NAVER_USER = 네이버 계정(@naver.com 포함)
NAVER_PASS = 네이버 비밀번호
```
